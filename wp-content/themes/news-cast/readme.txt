News Cast is the elementor based news theme gives perfect layouts to your news websites. Specifically made for branded news websites built with WordPress. It perfectly integrates with Elementor and Gutenberg page builders - you can manage your website pages with popular website builders in this theme. It has advanced features like sticky header, menu hover effects, category colors, social icons options, show or hide post elements, header toggle sidebars option, site layouts, footer columns, archive settings and many more. You must activate Elementor Plugin to get all the widgets provided by theme. Theme provides 7 elementor widgets Social Icons, Post Grid, Post List, Post Grid Alter, Post Carousel, Post Featured and author Box. Although, its homepage section are build with elementor page builder it has numerous generic settings in customizer. You can manage social icons, site colors, menu styling, category colors, archive and single pages layouts, header settings and footer settings in cutomizer. We have well-documented theme documentation available here https://doc.blazethemes.com/news-cast/. For easy user guide we have video tutorials on demo import and homepage setup : https://blazethemes.com/theme/news-cast. Live Demo URL: https://demo.blazethemes.com/news-cast/

== Installation ==
1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the news-cast.zip file. Click Install Now.
3. Click Activate to use your new theme right away.
